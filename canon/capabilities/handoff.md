# Handoff

## Purpose

Preserve continuity across multiple sessions, contributors, and assistants.

## Responsibilities

- capture git state
- summarize current work
- record blockers and next steps
- record deploy and environment state
- update shared session surfaces

## Key Question

Can someone restart from here without relying on chat memory?
