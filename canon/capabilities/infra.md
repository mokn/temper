# Infra

## Purpose

Operational review for environments, services, deploy pipelines, recovery, and system plumbing.

## Responsibilities

- environment discipline
- deploy plumbing
- background services
- indexers
- relayers
- telemetry pipelines
- rollback and recovery
- secrets and config hygiene

## Key Question

What can break in operation, and how fast can we recover when it does?
