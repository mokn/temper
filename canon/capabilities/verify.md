# Verify

## Purpose

Read-only verification that intended game state matches actual project and environment state.

## Responsibilities

- source-of-truth parity
- environment sanity
- data drift checks
- deploy-state checks
- generated artifact sanity

## Key Question

Does reality still match the intended source of truth?
